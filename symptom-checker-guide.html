Build the AI / NLP Model

This is the core of your project. Train a classifier to map symptoms → diseases.
💡

Start simple. Use a symptom-disease CSV dataset first with a Random Forest classifier. Once it works, layer NLP on top to parse free-text symptoms. Don't try to fine-tune BERT on day 1.

A

Step A: Load and Explore Dataset

Use the Kaggle Disease Symptom Prediction dataset. It contains 4,920 rows of symptoms mapped to 41 diseases.

Python
import pandas as pd

df = pd.read_csv('data/symptom_disease.csv')
print(df.head())
print(df['Disease'].value_counts())  # Check distribution
print(df.isnull().sum())               # Check missing values
B

Step B: Preprocess Symptoms

Encode symptoms as binary features (1 = present, 0 = absent). This turns each row into a vector the model can learn from.

Python
from sklearn.preprocessing import LabelEncoder

# Fill NaN with 0, encode symptom columns
df = df.fillna(0)

# Separate features and target
X = df.drop('Disease', axis=1)
y = df['Disease']

# Encode disease labels to numbers
le = LabelEncoder()
y_encoded = le.fit_transform(y)
C

Step C: Train Classifier

Random Forest is a strong starting point — accurate, interpretable, and fast to train. Aim for 90%+ accuracy on the test set.

Python
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score
import pickle

X_train, X_test, y_train, y_test = train_test_split(
    X, y_encoded, test_size=0.2, random_state=42
)

model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X_train, y_train)

acc = accuracy_score(y_test, model.predict(X_test))
print(f"Accuracy: {acc:.2%}")  # Target: > 90%

# Save model + label encoder
with open('model.pkl', 'wb') as f:
    pickle.dump({'model': model, 'encoder': le}, f)
D

Step D: Add NLP for Free-Text Input

Use spaCy to extract symptom keywords from natural sentences like "I have a fever and sore throat" and map them to the binary feature vector.

Python
import spacy

nlp = spacy.load("en_core_web_sm")
KNOWN_SYMPTOMS = list(X.columns)  # Your symptom list

def extract_symptoms(text: str) -> list:
    doc = nlp(text.lower())
    found = []
    for sym in KNOWN_SYMPTOMS:
        if sym.replace('_', ' ') in text.lower():
            found.append(sym)
    return found

def build_feature_vector(symptoms: list) -> list:
    vec = [1 if s in symptoms else 0 for s in KNOWN_SYMPTOMS]
    return vec

5

Backend API (FastAPI)

Create the REST endpoint that the frontend calls to get predictions.
Python — FastAPI main.py
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from pydantic import BaseModel
from predict import get_predictions

app = FastAPI(title="Symptom Checker API")

# Allow requests from React frontend
app.add_middleware(CORSMiddleware, allow_origins=["*"], allow_methods=["*"])

class SymptomRequest(BaseModel):
    symptoms: str   # e.g., "fever, headache, sore throat"

@app.post("/predict")
async def predict(req: SymptomRequest):
    predictions = get_predictions(req.symptoms)
    return {
        "predictions": predictions,   # Top 3 diseases + confidence
        "disclaimer": "For informational use only. Consult a doctor."
    }

# Run: uvicorn main:app --reload
⚠️

Always add CORS middleware or your React frontend won't be able to call the Python API. Also make sure to return a disclaimer field in every API response — evaluators will look for this.
